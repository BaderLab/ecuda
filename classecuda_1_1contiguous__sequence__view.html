Detailed Description

template<typename T>
class ecuda::contiguous_sequence_view< T >

View of data sequence residing in contiguous memory given a pointer and size.

This is a subclass of sequence_view that imposes the requirement that the underlying pointer refers to a contiguous block of memory. Thus, PointerType (the second template parameter of sequence_view) is strictly defined as a naked pointer of type T*.

This allows the assign method to be made available safely.
