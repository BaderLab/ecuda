template<typename T, typename PointerType = typename ecuda::reference<T>::pointer_type, std::size_t PaddingUnitBytes = sizeof(T)>
HOST DEVICE ecuda::padded_ptr< T, PointerType, PaddingUnitBytes >::padded_ptr ( pointer  p = pointer(),
const size_type  data_length = 1,
const size_type  padding_length = 0,
const size_type  pointer_position = 0 
)
inline

Default constructor.

The pointer p points to a location within a block of padded memory where every data_length contiguous elements of type T are followed by a block of ignorable memory that is padding_length units of size PaddingUnitBytes. pointer_position specifies the current location of the pointer within a data block.

Parameters
pa pointer to an element of type T
data_lengththe number of contiguous elements of type T that comprise the data block
padding_lengththe number of PaddingUnitBytes-sized elements of padding following each data block
pointer_positionthe index of the pointer within the current data block
