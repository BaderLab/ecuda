Detailed Description

template<typename T, typename PointerType = typename ecuda::reference<T>::pointer_type>
class ecuda::striding_ptr< T, PointerType >

A specialized pointer to memory holding a sequence of data with fixed-sized spacing between each value.

Strided memory is a block of contiguous memory where elements are separated by fixed-length padding. Thus, one has to "stride" over the padding to reach the next element. The term was borrowed from the GNU Scientific Library.
